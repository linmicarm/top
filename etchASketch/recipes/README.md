# 🍱 recipes

a handcrafted recipe collection page built with html and css, featuring a curated set of japanese dishes and drinks. the focus was on clean layout, warm aesthetics, and presenting recipes in a way that feels as good as the food itself.

---

## 🧠 concepts practiced

- flexbox layout for recipe cards and page structure
- semantic html for accessible, well-structured content
- css custom properties for consistent color theming
- typography and visual hierarchy for readability
- image handling and aspect-ratio consistency
- responsive design considerations

---

## 🛠️ how it's made

**tech used:** html, css

this is a static recipe collection page built section by section using no frameworks, no shortcuts. each recipe card was approached individually to keep the layout intentional and the content easy to browse. the visual style draws from the calm, minimal aesthetic of japanese food photography.

## ⚡ optimizations

if i were to revisit this project, i'd:

- make the layout **fully responsive** so recipe cards reflow gracefully on mobile
- add a **filter or search** feature to sort by ingredient or dish type
- improve **accessibility** with better semantic elements, alt text, and aria labels
- lazy-load images for faster initial page load
- add a print stylesheet so recipes look clean on paper

---

## 💭 lessons learned

- food photography does a lot of the heavy lifting. choosing the right images sets the whole tone of the page
- keeping a consistent card structure made the css much easier to reason about
- small typographic choices (line height, font pairing) matter more than expected when the content is text-heavy

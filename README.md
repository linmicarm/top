# The Odin Project
 
My assignments, projects, and exercises completed through The Odin Project, a free, open-source full-stack curriculum.
 
## How It's Made
 
**Tech used:** JavaScript, HTML, CSS
 
The Odin Project is a self-paced curriculum that doesn't hand you answers. It points you in the right direction and expects you to figure things out. This repo reflects that process: working through assignments independently, reading documentation, debugging on my own, and building projects from scratch.
 
Each section of the curriculum builds directly on the last, so this repo grows in complexity as the skills do.
 
## What's Inside
 
- **Foundations**: early exercises covering HTML, CSS, and JavaScript basics
- **Projects**: milestone builds assigned throughout the curriculum
- **Assignments**: smaller exercises focused on specific concepts or techniques

## Optimizations
 
Because TOP emphasizes understanding over speed, I focus on:
 
- Reading the official docs rather than jumping straight to tutorials
- Understanding *why* something works, not just *that* it works
- Refactoring projects after completing them to apply concepts learned later in the curriculum
## Lessons Learned
 
- Learning to learn is the most valuable skill a developer can build. TOP trains exactly that
- Struggling through a problem independently makes the solution stick far longer
- Documentation is your best friend once you stop being intimidated by it
- Progress feels slow day to day but significant when you look back week to week
